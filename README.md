# 🚗 台灣交通改革協作平台

[![GitHub issues](https://img.shields.io/github/issues/wy-zong/taiwan-road-reform)](https://github.com/wy-zong/taiwan-road-reform/issues)
[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

> 透過群眾協作，改善台灣道路設計，打造更安全的交通環境。

## 📋 專案簡介

這是一個開源的公民參與平台，讓所有人都能：

- 🔍 **回報問題**：發現有問題的道路設計？開個 Issue 告訴大家
- 🗺️ **查看地圖**：在互動地圖上瀏覽全台待改善的道路問題
- ✅ **記錄成果**：當問題被解決，透過 PR 歸檔成功案例

## 🚀 如何參與

### 回報道路問題

1. 點擊 **[Issues → New Issue](../../issues/new/choose)**
2. 選擇「🚗 道路問題回報」模板
3. 填寫地點、縣市、問題類型與描述
4. 上傳現場照片（強烈建議）
5. 送出！

### 協助確認與追蹤

- 瀏覽 [現有 Issues](../../issues)
- 如果你也觀察到相同問題，留言 `+1` 或補充資訊
- 協助向主管機關陳情（參考 [陳情教學](docs/how-to-petition.md)）

### 回報已解決的問題

當你發現某個問題已被改善：

1. Fork 這個 repo
2. 在 `resolved/` 資料夾建立檔案
3. 填寫改善資訊與照片
4. 提交 PR，並使用 `Closes #123` 關聯原 Issue
5. 等待 5 位參與者確認後合併

詳細規範請參考 [CONTRIBUTING.md](CONTRIBUTING.md)
